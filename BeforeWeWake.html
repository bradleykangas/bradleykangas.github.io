Before We Wake (2020):

Before We Wake is an unreleased project by Black Fox Games. I was the 3D
modeler/animator for the project. My primary task was working with the project's 2D
artist to translate the 2D orthographic drawings of the bear to a game-ready 3D model.

I was also responsible for game design (weapon/upgrades system), programming,
scene construction, UI graphics.
